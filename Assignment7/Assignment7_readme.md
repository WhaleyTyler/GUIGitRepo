## Project Overview
This project is a GUI-based measurement converter built using Python and PySide6 (Qt for Python).
The application allows users to convert values between inches and meters using a simple and user-friendly interface.

The program includes input validation, error handling, and a clean layout with accessible design features.

---

## Development Tools
- Python 3.14.3
- PySide6 6.11.0
- IDE: PyCharm

---

## How to Run the Program
1. Ensure Python is installed on your system.
2. Install PySide6 (if not already installed)
```pip install Pyside6```
3. Navigate to the project folder in your terminal.
4. Run the program using:
```python converter.py```  
5. The application window will open and be ready for use.  

---

## Features
- Convert inches ↔ meters
- Input validation with error dialogs
- Clear and Exit buttons
- Accessible color theme
- Embedded image (house.png)
